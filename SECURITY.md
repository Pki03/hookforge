# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please open a GitHub Issue with the label `security`.
Do not post details publicly — include contact info so we can follow up privately.

We will acknowledge receipts within 48 hours and aim for a fix within 7 days.
